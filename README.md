# 🎮 Pixel Astro Blog

像素风 Astro 博客主题 — 功能对标 [vhAstro-Theme](https://github.com/uxiaohan/vhAstro-Theme)

## ✨ 特性

- 🎨 像素风格设计（Press Start 2P + VT323 字体）
- 📱 响应式两列布局
- ⌨️ 首页打字机效果
- 📝 文章卡片（封面/摘要/阅读时间/字数）
- 🔍 本地搜索
- 💬 评论系统（Twikoo / Waline）
- 🏷️ 标签 / 分类 / 归档
- 💭 动态 / 🌍 圈子 / 🤝 友链 / ✉️ 留言板
- ☕ 赞赏功能 / 📜 版权声明
- 📡 RSS / Sitemap
- 🚀 GitHub Pages 自动部署
- 🎭 404 像素风页面

## 🚀 快速开始

```bash
git clone https://github.com/你的用户名/pixel-astro-blog.git
cd pixel-astro-blog
pnpm install
pnpm dev
```

## 🚀 部署到 GitHub Pages

1. 创建 GitHub 仓库
2. 推送代码到 `main` 分支
3. Settings → Pages → Source 选择 "GitHub Actions"
4. 自动构建部署！
