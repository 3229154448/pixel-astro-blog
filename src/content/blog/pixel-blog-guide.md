## 🎮 为什么选择像素风？

像素风（Pixel Art）是一种经典的视觉风格，起源于早期电子游戏。它用最少的像素表达最丰富的信息，有一种独特的复古美感。

## 🛠️ 技术选型

- **Astro** — 现代静态站点生成器，速度快、体积小
- **TypeScript** — 类型安全，开发体验好
- **Less** — CSS 预处理器
- **Press Start 2P** — 经典像素字体
- **VT323** — 终端风格字体

## 📐 像素风设计要点

### 边框与阴影

像素风的核心是**硬阴影**和**粗边框**：

```css
.card {
  border: 3px solid #333c57;
  box-shadow: 4px 4px 0 0 #000;
}
```
